# restaurant-rating-predictor
ML model to predict restaurant ratings using Decision tree regression | R² = 0.92
## 📊 Results
- **Best Model:** Decision Tree Regression
- **R² Score:** 0.92 (92% accuracy)
- **RMSE:** 0.417
- **Key Finding:** Votes is the #1 predictor of restaurant ratings (94.8% importance)
